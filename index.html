Password Lock System

A security system that only gives access to the user entered the correct password and denies access when an incorrect password is entered.

    Specifications

  • 230VAC or 12VDC supply
  • 5 characters long password
  • 2 modes (blocking & try again)
  • Changeable password

    Applications

  • Can be used for security to avoid unwanted visitors.
  • Can be used as a lock for privacy to prevent people from having access to your things without your permission.
  • Can be used for safety in engineerign industry to only the authorised to operate on the systems.

Obstacle Avoiding Password Lock System

A security system that only gives access to the user entered the correct password and denies access when an incorrect password is entered and avoids hitting an obstacle when closing the gate.

    Specifications

  • 230VAC or 12VDC supply
  • 5 characters long password
  • 2 modes (blocking & try again)
  • Changeable password
  • Detects up to 4m

    Applications

  • Can be used to open and block entrance in any premises.

Smart Filling and Pricing System

A systems that fills containers with liquid and calculates the price according to the amount of litres filled.

    Specifications

  • 230VAC or 12VDC supply
  • Can fill up to 2L
  • Can fill manually
  • Fills 100ml/s

    Applications

  • Can be used in shop for selling drinks
  • Can be used for filling paraffin, petrol and more.

Smart LV Distribution System

A systems that can be controlled manually or remotely just by sending an SMS.

    Specifications

  • 230VAC Single Phase System
  • 12VDC Backup Control System
  • 4 Distribution outgoing panel feeders + 1 Main Feeder

    Applications

  • Easy control in low voltage electrical distribution system.
  • Can be used in hostels or school residences to easily control each outgoing feeder.

DOL Single Phase Motor Control

A control system that uses START/STOP buttons and make it possible for the motor to run in FORWARD/REVERSE direction.

    Specifications

  • 230VAC Supply
  • Rotation: Forward & Reverse

    Applications

  • Can be used in conveyor belts
  • Can be used in escalator systsem
  • and more
