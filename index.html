WHO ARE WE?
Sinelutec is derived from the name of the founder, Sinethemba Lusawana and is a company whose main objective is to teach and share knowledge with each and everyone interested in electrical engineering.

OUR VISION?
We want to see capable and knowledgable young electrical engineering students and electricians designing their own electrical projects to solve real world problems.

WHAT WE DO?
Before a project is implemented it is first built and simulated on softwares like Proteus, build it on a breadboard and finally build it on the PCB or anything similar.

In all that we do we follow safety precautions and necessary electrical standards when working with electricity in our electrical systems.

                                    /*
                                      Fade
                                    
                                      This example shows how to fade an LED on pin 9 using the analogWrite()
                                      function.
                                    
                                      The analogWrite() function uses PWM, so if you want to change the pin you're
                                      using, be sure to use another PWM capable pin. On most Arduino, the PWM pins
                                      are identified with a "~" sign, like ~3, ~5, ~6, ~9, ~10 and ~11.
                                    
                                      This example code is in the public domain.
                                    
                                      https://www.arduino.cc/en/Tutorial/BuiltInExamples/Fade
                                    */
                                    
                                    int led = 9;           // the PWM pin the LED is attached to
                                    int brightness = 0;    // how bright the LED is
                                    int fadeAmount = 5;    // how many points to fade the LED by
                                    
                                    // the setup routine runs once when you press reset:
                                    void setup() {
                                      // declare pin 9 to be an output:
                                      pinMode(led, OUTPUT);
                                    }
                                    
                                    // the loop routine runs over and over again forever:
                                    void loop() {
                                      // set the brightness of pin 9:
                                      analogWrite(led, brightness);
                                    
                                      // change the brightness for next time through the loop:
                                      brightness = brightness + fadeAmount;
                                    
                                      // reverse the direction of the fading at the ends of the fade:
                                      if (brightness <= 0 || brightness >= 255) {
                                        fadeAmount = -fadeAmount;
                                      }
                                      // wait for 30 milliseconds to see the dimming effect
                                      delay(30);
                                    }
