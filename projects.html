Password Lock System

A security system that only gives access to the user entered the correct password and denies access when an incorrect password is entered.

    Specifications

  • 230VAC or 12VDC supply
  • 5 characters long password
  • 2 modes (blocking & try again)
  • Changeable password

Obstacle Avoiding Password Lock System

A security system that only gives access to the user entered the correct password and denies access when an incorrect password is entered and avoids hitting an obstacle when closing the gate.

    Specifications

  • 230VAC or 12VDC supply
  • 5 characters long password
  • 2 modes (blocking & try again)
  • Changeable password
  • Detects up to 4m

Smart Filling and Pricing System

A systems that fills containers with liquid and calculates the price according to the amount of litres filled.

    Specifications

  • 230VAC or 12VDC supply
  • Can fill up to 2L
  • Can fill manually
  • Fills 100ml/s
